<h2>Details</h2>
<a id="site_1mp"></a>
<a id="main-page"></a>
<h3>Main Page</h3>
<span class="inline-text">Main page is generated by Doxygen as indexpage.xml. It needs to be defined somewhere in the project&apos;s source code </span>
<code class="typewriter">\mainpage</code>
<a id="site_1index"></a>
<a id="index"></a>
<h3>Index</h3>
<span class="inline-text">Index implements a project wide table of contents. It is sorted by pages, directories, namespaces, and classes.</span>
<a id="site_1ni"></a>
<a id="namespace-index"></a>
<h3>Namespace Index</h3>
<span class="inline-text">The namespace index is defined in </span>
<code class="typewriter">namespace_index</code>
<span class="inline-text">. It contains a List of all namespaces defined in the current index.xml.</span>
<a id="site_1ci"></a>
<a id="class-index"></a>
<h3>Class Index</h3>
<span class="inline-text">The class index is defined in </span>
<code class="typewriter">class_index</code>
<span class="inline-text">. It contains a List of all classes defined in the current index.xml.</span>
<a id="site_1pi"></a>
<a id="page-index"></a>
<h3>Page Index</h3>
<span class="inline-text">The page index is defined in </span>
<code class="typewriter">page_index</code>
<span class="inline-text">. It contains a List of all pages defined in the current index.xml.</span>
<a id="site_1di"></a>
<a id="directory-index"></a>
<h3>Directory Index</h3>
<span class="inline-text">The directory index is defined in </span>
<code class="typewriter">directory_index</code>
<span class="inline-text">. It contains a List of all top level directories in the current project.</span>
<br/>
<br/>
<span class="italic-text"><i>project refers to the current index.xml file.</i></span>
