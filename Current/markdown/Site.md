<a id="details"></a>
<h2>Details</h2>
<img src="../images/dot/internal-diagram-1.dot.svg"/><br/>
<span class="inline-text">MainPage is the site root. It leads into a section index page. Each section index page is split into separate files. Each file provides an alphabetical list of its contents.</span>
<br/>
<img src="../images/class.svg"/><a href="classMdDox_1_1IndexPageWriter.md#mddoxindexpagewriter">MdDox::IndexPageWriter</a>
<br/>
<span class="inline-text">
Defines the layout for the primary index pages.</span>
<br/>
<img src="../images/class.svg"/><a href="classMdDox_1_1NamespacePageWriter.md#mddoxnamespacepagewriter">MdDox::NamespacePageWriter</a>
<br/>
<span class="inline-text">
 Subdivies the page into members of the namespace.</span>
<br/>
<img src="../images/class.svg"/><a href="classMdDox_1_1ClassPageWriter.md#mddoxclasspagewriter">MdDox::ClassPageWriter</a>
<br/>
<span class="inline-text">
 Subdivies the page into members of the class.</span>
<br/>
<img src="../images/class.svg"/><a href="classMdDox_1_1DirectoryPageWriter.md#mddoxdirectorypagewriter">MdDox::DirectoryPageWriter</a>
<br/>
<span class="inline-text">
 Allows browsing all files that have been processed.</span>
<br/>
<img src="../images/class.svg"/><a href="classMdDox_1_1GenericPageWriter.md#mddoxgenericpagewriter">MdDox::GenericPageWriter</a>
<br/>
<span class="inline-text">
 Defines a page from only a brief and detail description.</span>
<br/>
<span class="inline-text">
 Use </span>
<img src="../images/class.svg"/><a href="classMdDox_1_1PageWriter.md#mddoxpagewriter">MdDox::PageWriter</a>
<span class="inline-text"> to derive a new page type.</span>
<a id="Site_1MainPage"></a>
<a id="mainpage"></a>
<h3>MainPage</h3>
<span class="inline-text">Main page is generated by Doxygen as indexpage.xml. It needs to be defined somewhere in the project&apos;s source code </span>
<code class="typewriter">\mainpage</code>
<a id="Site_1Index"></a>
<a id="index"></a>
<h3>Index</h3>
<span class="inline-text">The index implements a project wide table of contents. It is sorted by pages, directories, namespaces, and classes.</span>
<a id="Site_1NamespaceIndex"></a>
<a id="namespaceindex"></a>
<h3>NamespaceIndex</h3>
<span class="inline-text">The namespace index is defined in </span>
<code class="typewriter">namespace_index</code>

<a id="classindex"></a>
<h3>ClassIndex</h3>

<a id="pageindex"></a>
<h3>PageIndex</h3>
