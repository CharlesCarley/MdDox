<a id="main-page"></a>
<h4>Main Page</h4>
<span class="inline-text">Main page is generated by Doxygen as indexpage.xml. It needs to be defined somewhere in the project&apos;s source code with the Doxygen special command </span>
<code class="typewriter">\mainpage</code>
<a id="index"></a>
<h4>Index</h4>
<span class="inline-text">Index implements a project wide table of contents. It is sorted by page, directory, namespace, and then class index.</span>
<a id="namespace-index"></a>
<h4>Namespace Index</h4>
<span class="inline-text">The namespace index is defined in the </span>
<code class="typewriter">namespace_index</code>
<code class="typewriter">.{outputFileEx}</code>
<span class="inline-text">. It contains a list of all namespaces defined in the current project source.</span>
<a id="class-index"></a>
<h4>Class Index</h4>
<span class="inline-text">The class index is defined in </span>
<code class="typewriter">class_index</code>
<code class="typewriter">.{outputFileEx}</code>
<span class="inline-text">. It contains a list of all classes defined in the current project source.</span>
<a id="page-index"></a>
<h4>Page Index</h4>
<span class="inline-text">The page index is defined in </span>
<code class="typewriter">page_index</code>
<code class="typewriter">.{outputFileEx}</code>
<span class="inline-text">. It contains a list of all pages defined in the current project source.</span>
<a id="directory-index"></a>
<h4>Directory Index</h4>
<span class="inline-text">The directory index is defined in </span>
<code class="typewriter">directory_index</code>
<code class="typewriter">.{outputFileEx}</code>
<span class="inline-text">. It contains a list of all top level directories in the current project. </span>
